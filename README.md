# Data Science Jobs Dashboard (Excel Project)

## Introduction

This project analyzes the **global data science job market** using a dataset of over 30,000 job postings from 2023.

The goal of this project is to provide a clear, data-driven overview of:
- Different data roles  
- Salary trends  
- Job locations  
- Hiring platforms  

This dashboard is designed to help:
- Job seekers  
- Career switchers  
- Beginners in data analytics  

understand the data job landscape in a simple and interactive way.

---

## Dataset

The dataset contains information about 10 major data-related roles, including:

- Data Analyst  
- Data Scientist  
- Data Engineer  
- Machine Learning Engineer  
- Business Analyst  
- Software Engineer  
- Cloud Engineer  

It includes details such as:
- Salary (yearly)  
- Country  
- Job type (Full-time, Part-time, etc.)  
- Hiring platform  
- Work-from-home availability  
- Skills required  

---

## Key Insights

- Senior roles like **Senior Data Scientist and Machine Learning Engineer** offer the highest salaries  
- Most data-related jobs are **full-time roles**  
- Job opportunities are highly concentrated in **countries like the United States**  
- Certain platforms dominate job postings (e.g., Ladders)
